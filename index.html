Portfolio

Canvas paint

Canvas Paint is a simple and intuitive web-based drawing application built using HTML5. It allows users to create and edit drawings directly within their browser, offering various tools like brushes, colors, and shapes for a seamless digital painting experience. The app provides basic functionalities such as freehand drawing, line drawing, color picking, erasing, and clearing the canvas. Designed with a clean and user-friendly interface, Canvas Paint is perfect for casual doodling or creating quick sketches. It is fully responsive and works across different devices, ensuring accessibility on both desktop and mobile platforms.

Travel website

Travel Explorer is a dynamic and user-friendly travel website designed to help users discover, plan, and book their dream vacations. With a sleek and responsive design, the website provides a seamless browsing experience across all devices. It features detailed destination guides, curated travel itineraries, and an integrated booking system for flights, hotels, and tours. Users can explore popular tourist spots, hidden gems, and read reviews to make informed decisions. Travel Explorer also offers personalized recommendations based on user preferences, helping travelers create customized trips tailored to their interests. Whether you're a solo traveler or planning a family vacation, Travel Explorer makes your travel planning effortless and enjoyable.
